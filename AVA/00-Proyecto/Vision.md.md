# Visión del Proyecto AVA

AVA (Aluminios y Vidrios Arriaga) es una plataforma web y móvil diseñada para digitalizar la operación completa del negocio.

La plataforma permitirá controlar:

- Inventario
- Catalogo de clientes y proveedores
- Cotizaciones
- Ventas
- Agenda
- Instalaciones
- Producción
- Tareas
- Gastos
- Ingresos

Objetivo principal:

Reducir errores operativos, acelerar la generación de cotizaciones y tener control total de las operaciones desde cualquier dispositivo.

Usuarios principales:

- Administrador (Dueño)
- Secretaria 
- Trabajadores

Beneficios esperados:

- Menos tiempo en cotizaciones
- Menos pérdidas de material
- Mejor control de inventario
- Seguimiento de clientes
- Seguimiento de proveedores
- Control de tareas
- Control de trabajos pendientes
- Información financiera básica
