**Cliente solicita cotización**
`Cliente -> Secretaria/jefe -> agendar visita de domicio o registro de medidas -> calculadora -> generar cotización -> cliente acepta -> Venta (anticipo) -> Orden de trabajo -> fabricación ->  Instalación -> Entrega de trabajo -> Venta (pago final)`

**Compra de material**
`Stock bajo -> contactar proveedor -> realizar solicitud de compra -> recepción -> Inventario`

**Calculadora de despiece** 
`Obtener medidas finales -> calcular medidas de producto x pieza -> asignar tarea -> realizar cortes de material -> realizar armado -> finalizar tarea`

**Asignar viaje de instalación**
Asignar fecha de instalación -> Avisar al cliente -> realizar instalación

Venta rápida
Cliente realiza pedido -> se realiza cobro -> se entrega producto

Asignar viaje de medidas
Cliente se contacta -> se asigna fecha y hora de visita -> se realiza viaje
