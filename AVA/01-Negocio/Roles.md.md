# Roles

## Administrador

Responsabilidades:

- Gestionar catálogo (crear - añadir - consultar - modificar - eliminar)
- Autorizar descuentos
- Revisar ventas
- Revisar gastos
- Asignar trabajos
- Registrar precios
- Registrar montos para cálculos de cotizaciones 

Permisos:

- Acceso total

---

## Secretaria

Responsabilidades:

- Registrar clientes
- Registrar proveedores
- Crear cotizaciones
- Registrar ventas
- Registrar compras
- Programar visitas
- Agendar pendientes en calendario
- Registrar facturas
- Corte de caja

Permisos:

- Operación comercial y administrativo

---

## Trabajador

Responsabilidades:

- Consultar tareas
- Consultar visitas pendientes
- Registrar avances (iniciada - proceso - terminada)
- Subir fotografías

Permisos:

- Solo información de tareas, pendientes y calendario
