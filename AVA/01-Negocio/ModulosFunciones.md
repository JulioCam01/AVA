**Login/Register**
	Campos obligatorios (email omitirlo)
	- Número celular
	- Contraseña
	- tipo de usuario (secretaria/trabajador)

* Usuario normal se puede registrar 
* Administrador debe de validar usuario para crearlo. 
* Modulo para admin para registrar usuarios manualmente.
* Modulo para dar de baja usuarios (no eliminar)

**Home**
	Tablero de tareas pendientes
	Tablero de tareas en proceso
	Tablero de tareas terminadas
	Calendario con las visitas e instalaciones programadas. 
(Todos los usuarios lo pueden ver)

**Catálogos** 
	- Registro de Clientes/Prospecto (datos personales, datos domicilio, tipo[cliente/prospecto])
	- Registro de Proveedores (datos fiscales, datos domicilio, datos bancarios)
	- Registro de productos (nombre, modelo, marca, categoría, descripción, fotografía, precio compra, precio venta, tipo de unidad [pieza, metro lineal, metro cuadrado, etc ], tipo de compra, tipo de venta)
	- Registro de usuarios --Solo admin

**Registro de tareas**
	- Registrar tareas (tipo de tarea, fecha de inicio, usuario asignado, etapa de tarea)
	- Asignar tarea
	- Cancela tarea (solo admin)
	- 
	[
		* Tipo de tarea: ir a medir, ir a instalar, cortar vidrios, cortar aluminio, armar puertas/ventanas/domos, ordenar área de trabajo 
		* Etapa de tarea: pendientes, proceso, terminado, cancelado, sin asignar
	]

**Registrar eventos en calendario**
	- Registrar evento (tipo de evento, asignar cliente, fecha, hora, ubicación, descripción )
	- Modificar evento
	- Cancelar evento --Solo admin

**Cotizador**
	- Crear nota de venta (ventas rápidas= vidrios, productos, material etc.  )
	- Crear presupuesto (cotizar puertas, ventanas, canceles de baño, domos)
	- Asignar a cliente o publico en general
	- Asignar descuentos -- solo admin
	- Modificar presupuestos

**Calculadora de despiece**
	- Obtener medidas de corte para ventanas
	- Obtener medidas de corte para puertas
	- Obtener medidas de corte para canceles de baño

Archivo de Excel basico donde hacia cotizaciones simples y calculadora de despiece para 3 tipos de ventanas
